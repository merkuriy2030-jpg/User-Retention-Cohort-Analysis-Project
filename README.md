# User Retention & Cohort Analysis Project

## Project Overview
This project focuses on the end-to-end data analysis pipeline evaluating user retention and behavioral patterns. The primary goal was to analyze "messy" raw user data, calculate key retention metrics, and compare the performance of different acquisition channels (Organic vs. Promo) to drive data-driven marketing decisions.

## Tech Stack & Tools
* **SQL (PostgreSQL):** Data extraction, transformation, complex JOINs, and CTEs.
* **Google Sheets:** Pivot tables, conditional formatting, slicers, and interactive dashboards.

## Key Features & Workflow
1. **Data Cleaning & Parsing:** Cleaned inconsistent date formats, handled whitespaces, and standardized raw text inputs into valid timestamps using SQL.
2. **Cohort Extraction:** Wrote multi-stage SQL queries with Common Table Expressions (CTEs) to segment users by registration month and calculate monthly offsets.
3. **Retention Rate Calculation:** Developed formulas to compute Month-over-Month (MoM) Retention Rates.
4. **Dashboard Building:** Created production-ready cohort tables and dynamic data visualizations in Google Sheets.
5. **Business Insights:** Benchmarked organic traffic against promo campaigns and formulated recommendations for improving customer loyalty.

## Key Insights Delivered
* Identified core differences in retention curves between promo and organic cohorts.
* Provided actionable recommendations for retention marketing spend efficiency.
