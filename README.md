# User Retention & Cohort Analysis Project

### 🛠️ Technologies used:
* **SQL** (Data extraction, date parsing, table joins, cohort calculations)
* **Google Sheets** (Pivot Tables, conditional aggregation, heatmaps, conditional formatting)
* **Git & GitHub** (Version Control)

### 📝 General Description:
This project focuses on evaluating user retention using cohort analysis based on transactional and activity data. It combines power SQL queries for backend data extraction and transformation with Google Sheets for interactive visualization and business analysis.

### 💡 Project Idea:
The core idea is to segment users into cohorts based on their signup/acquisition date and analyze their retention dynamics over time (Retention Rate). This analysis helps evaluate acquisition quality and compare the long-term value of different marketing channels.

### 🎯 My Tasks:
* **Data Extraction & Cleaning:** Extracted, filtered, and cleaned raw event data from the database using SQL, parsing complex date formats and joining multiple tables.
* **Cohort Calculations:** Calculated user cohorts and monthly shifts (user tenure) directly via optimized SQL queries.
* **Dashboard Building:** Built interactive Cohort Tables in Google Sheets using Pivot Tables, advanced formulas, and conditional formatting (retention heatmaps).
* **Segment Analysis:** Analyzed and compared Retention Rates for two key user segments: Promo Users (acquired via marketing campaigns) vs. Organic Users.

### 📊 Project Result:
Successfully transformed thousands of database rows into an interactive, readable business dashboard. Formulated data-driven conclusions regarding acquisition quality (Acquisition Quality), showing which user group demonstrates higher loyalty, better retention, and superior long-term lifetime value.
